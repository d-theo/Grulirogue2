### Grulirogue 2

Rogue like inspired by Pixel Dungeon, Brogue, DCSS and my wife :) Work in progress !

## install locally

* npm install
* npm start

## Features

* 3 maps
* 5 weapons combinable with 9 enchants
* 2 armour combinable with 5 enchants
* 10 potions
* 6 scrolls
* 15 monsters
* 9 consumables
* 5 special rooms
* 2 boss
* 2 IA
